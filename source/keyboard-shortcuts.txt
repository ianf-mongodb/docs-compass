Keyboard Shortcuts

.. contents:: On this page

The following sections outline keyboard shortcuts for navigating MongoDB
Compass.


.. list-table::
   :widths: 20 50 30
   :header-rows: 1

   * - Category
     - Description

   * - General
     - Navigate to next tab.
     - `Tab`

   * - General
     - Navigate to previous tab.
     - `Shift + Tab`

   * - Menu

     - `CmdOrCtrl + Q`

   * - Menu

     - `CmdOrCtrl + ,`

   * - Menu
     - Hide window.
     - `Command + H`

   * - Menu

     - `Command + Shift + H`

   * - Menu

     - `CmdOrCtrl + N`

   * - Menu
     - Undo.
     - `Command + Z`

   * - Menu
     - Redo.
     - `Shift + Command + Z`

   * - Menu
     - Cut.
     - `Command + X`

   * - Menu
     - Copy.
     - `Command + C`

   * - Menu
     - Paste.
     - `Command + V`

   * - Menu

     - `Command + A`

   * - Menu
     - Find.
     - `CmdOrCtrl + F`

     - `Alt + CmdOrCtrl + S`

     - `CmdOrCtrl + Shift + ]`

   * - Workspace

     - Close current tab.
