==================
Keyboard Shortcuts
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The following table outlines keyboard shortcuts for navigating MongoDB
Compass.

.. list-table::
   :widths: 20 50 30
   :header-rows: 1

   * - Category
     - Description
     - Command
   
   * - General
     - Navigate to next tab.
     - `Tab`

   * - General
     - Navigate to previous tab.
     - `Shift + Tab`

   * - Menu
     - Quit application.
     - `CmdOrCtrl + Q`

   * - Menu
     - Show settings.
     - `CmdOrCtrl + ,`

   * - Menu
     - Hide window.
     - `Command + H`

   * - Menu
     - Hide other windows.
     - `Command + Shift + H`

   * - Menu
     - New connection.
     - `CmdOrCtrl + N`

   * - Menu
     - Undo.
     - `Command + Z`

   * - Menu
     - Redo.
     - `Shift + Command + Z`

   * - Menu
     - Cut.
     - `Command + X`

   * - Menu
     - Copy.
     - `Command + C`

   * - Menu
     - Paste.
     - `Command + V`

   * - Menu
     - Select all.
     - `Command + A`

   * - Menu
     - Find.
     - `CmdOrCtrl + F`

   * - Menu
     - Online help.
     - `F1`

   * - Menu
     - Share schema as JSON.
     - `Alt + CmdOrCtrl + S`

   * - Menu
     - Reload screen.
     - `CmdOrCtrl + Shift + R`

   * - Menu
     - Reload data.
     - `CmdOrCtrl + R`

   * - Menu
     - Toggle sidebar.
     - `CmdOrCtrl + Shift + D`

   * - Menu
     - Actual size.
     - `CmdOrCtrl + 0`

   * - Menu
     - Zoom in.
     - `CmdOrCtrl + =`

   * - Menu
     - Zoom out.
     - `CmdOrCtrl + -`

   * - Menu
     - Toggle DevTools.
     - `Alt + CmdOrCtrl + I`

   * - Menu
     - Minimize.
     - `Command + M`

   * - Workspace
     - Navigate to next tab.
     - `CmdOrCtrl + Shift + ]`

   * - Workspace
     - Navigate to previous tab.
     - `CmdOrCtrl + Shift + [`

   * - Workspace
     - Close current tab.
     - `CmdOrCtrl + W`

   * - Workspace
     - Open new tab.
     - `CmdOrCtrl + T`

   * - Aggregation Focus Mode
     - Add a new stage after current one.
     - `CmdOrCtrl + Shift + B`

   * - Aggregation Focus Mode
     - Add a new stage before current one.
     - `CmdOrCtrl + Q`

   * - Aggregation Focus Mode
     - Navigate to next stage.
     - `CmdOrCtrl + Shift + 9`

   * - Aggregation Focus Mode
     - Navigate to previous stage.
     - `CmdOrCtrl + Shift + 0`

   * - Editors
     - Copy.
     - `CmdOrCtrl + C`

   * - Editors
     - Paste.
     - `CmdOrCtrl + P`

   * - Editors
     - Cut.
     - `CmdOrCtrl + V`

   * - Editors
     - Comment out code.
     - `CmdOrCtrl + /`

   * - Editors
     - Prettify code.
     - `Ctrl + Shift + B`

   * - Editors
     - Copy all.
     - `Ctrl + Shift + C`

   * - Editors
     - Undo.
     - `CmdOrCtrl + Z`

   * - Editors
     - Redo.
     - `CmdOrCtrl + Shift + Z`

   * - Query bar
     - Submit query.
     - `Enter (from the query bar)`

   * - Web Shell
     - Toggle shell.
     - `Ctrl + `````

   * - Web Shell
     - Submit query.
     - `Enter (from the query bar)`

   * - Web Shell
     - Deletes the next character.
     - `Ctrl` + `D`

   * - Web Shell
     - Moves the cursor to the end of the line.
     - `Ctrl` + `E`

   * - Web Shell
     - Moves the cursor Forward one character.
     - `Ctrl` + `F`

   * - Web Shell
     - Erases one character, similar to hitting backspace.
     - `Ctrl` + `H`

   * - Web Shell
     - Clears the screen, similar to the clear command.
     - `⌘` + `L`

   * - Web Shell
     - Swap the last two characters before the cursor.
     - `Ctrl` + `T`

   * - Web Shell
     - Changes the line to Uppercase.
     - `Ctrl` + `U`

   * - Web Shell
     - Cycle backwards through command history.
     - `↑`

   * - Web Shell
     - Cycle forwards through command history.
     - `↓`
