.. _compass-bulk-update:

=========================
Modify Multiple Documents
=========================

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can perform bulk update operations on multiple documents in Compass
by using the :guilabel:`Update Documents` modal. Performing updates with
the :guilabel:`Update Documents` modal helps you visualize updates 
to your data before you apply them.

About this Task
---------------

- Any syntax that works with the ``update`` parameter of the
  :method:`db.collection.updateMany()` is valid. 
  
- The :guilabel:`Update Documents` modal does not support any ``options`` 
  parameters such as upsert, writeConcern or collation.

Before you Begin
----------------

The :guilabel:`Update Documents` modal is available starting
in Compass version ``1.5.X``. For instructions on updating or installing
the latest Compass version, see :ref:`download-install`. 

Steps
-----

.. procedure::
   :style:  normal

   .. step:: Open the Bulk Update Modal

      From the :guilabel:`Documents` tab, click the 
      :icon-lg:`Edit` :guilabel:`Update` button.

      The :guilabel:`Update Documents` modal displays. The following 
      table summarizes the UI of the modal:

      .. list-table::
         :widths: 40 60
         :header-rows: 1

         * - UI Element
           - Description

         * - :guilabel:`Filter`
           - Any filter criteria specified on the 
             :guilabel:`Query Bar` is applied to the 
             :guilabel:`Update Documents` modal.
         * - :guilabel:`Update`
           - This is the update syntax that is applied to the 
             documents specified. Any syntax that works with the 
             ``update`` parameter of the
             :method:`db.collection.updateMany()` is valid.
         * - :guilabel:`Preview`
           - A preview of documents after the update syntax is applied.

   .. step:: Enter the update syntax 

      a. In the :guilabel:`Update` text field, provide the update 
         syntax.

         The number of documents affected by the update displays in 
         the top of the :guilabel:`Update Documents` modal in 
         parentheses.

      #. Click :guilabel:`Update Documents`.

         Compass applies the :guilabel:`Update` syntax to the documents
         within the filter criteria in the :guilabel:`Filter` expression.

         .. note::

            A sample of documents displays under the :guilabel:`Preview` 
            header. The documents in :guilabel:`Preview` show how the 
            :guilabel:`Update` syntax affects documents in your 
            collection.

   .. step:: (Optional) Name and save your update syntax

         You can name and save the update syntax in the 
         :guilabel:`Update Documents` modal. Saving your syntax adds it 
         to your favorite queries for that collection and allows you 
         to load and copy the syntax after you close the modal.

         a. Click the :guilabel:`Save` button on the bottom-left of the 
            modal.
         #. Enter a name for the update syntax.
         #. Click the green :guilabel:`Save` button. A popup toast 
            message displays, indicating the statement was saved.

            For details on how to access favorited queries, see 
            :ref:`saved-favorite-query-view`.

Example
-------

The following examples use the 
:ref:`sample_mflix dataset <sample-data>`. 

This example updates the ``tomatoes.viewer.numReviews`` and 
``tomatoes.viewer.meter`` fields with the Compass 
:guilabel:`Update Documents` modal.

Apply a filter in the :guilabel:`Query Bar` to filter movies which
have a ``year`` of ``1917``.

.. code-block:: javascript

   { 'year' : 1917 }

Click the :guilabel:`Update` button, the :guilabel:`Update Documents` 
modal displays.

In the :guilabel:`Update` text box, paste the following syntax:

.. code-block:: javascript

   { 
      $inc: { "tomatoes.viewer.numReviews" : 1}, 
      $set: { "tomatoes.viewer.meter" : 99 } 
   }

This syntax:

- Increments the ``tomatoes.viewer.numReviews`` field by ``1``.
- Sets the ``tomatoes.viewer.meter`` field to ``99``.

The :guilabel:`Preview` section populates with sample documents
that are affected by the update syntax.

To preview the updates to the :guilabel:`numReviews` 
and :guilabel:`meter` fields: 

- Click the :icon-fa5:`angle-right` arrow icon next to 
  :guilabel:`tomatoes`. 
- Click the :icon-fa5:`angle-right` arrow icon next to 
  :guilabel:`rating`.

Click the :guilabel:`Update Documents` button to update the collection's
data.
