================================
Enable Natural Language Querying
================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1

You can enable the Compass :guilabel:`AI Query Generator` to assist you 
in writing MongoDB queries and aggregations. The 
:guilabel:`AI Query Generator` can provide you with a starting point 
for your queries and help you learn how to write MQL statements.

About this Task
---------------

You can alternatively enable the :guilabel:`AI Query Generator` 
when prompted by the use natural language call to action modal. This modal
displays after updating to the latest version of Compass that supports
natural language querying.

Before you Begin
----------------

You must connect to an Atlas deployment to use 
the :guilabel:`AI Query Generator`. For details on how to get a free
Atlas account and deploy a free M0 cluster, see :ref:`atlas-getting-started`.
